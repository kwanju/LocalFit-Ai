# ADR-012: VAD로 silero-vad 채택

- **상태**: Accepted
- **작성일**: 2026-05-21
- **관련 ADR**: ADR-005 (STT)

## 컨텍스트

S2S/S2C 모드에서 사용자 발화 시작·종료를 감지해야 한다. 운동 중 거친 호흡, 옷 마찰음, 헬스장 배경 소음 환경을 견뎌야 한다.

## 결정

**silero-vad**를 채택한다. PyTorch 기반, 1MB 미만 모델.

설치: `pip install silero-vad`

## 결과

### 긍정
- webrtcvad 대비 노이즈 환경 정확도 우수
- Python 통합 단순
- CPU에서도 실시간 동작 — GPU 자원 절약

### 부정
- 가벼운 PyTorch 종속성 추가 (이미 Whisper로 로드됨)
- 운동 호흡 소리에 대한 별도 튜닝 필요할 수 있음 (P0 후반 검증)

## 대안

| 후보 | 탈락 사유 |
|---|---|
| webrtcvad (PRD 원안) | 호흡·신음·기계음 환경에서 false trigger 잦음 |
| Picovoice Cobra | 상용, 무료 사용 제한 |
| WebRTC VAD via aiortc | 헬스장 노이즈 환경에 약함 |
