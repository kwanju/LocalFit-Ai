# ADR-005: STT로 faster-whisper (large-v3-turbo) 채택

- **상태**: Accepted (Revised 2026-05-21)
- **이력**: 초기 medium 모델 결정에서 large-v3-turbo로 모델만 업그레이드. 어댑터 코드 무변경
- **관련 ADR**: ADR-010 (어댑터 인터페이스), ADR-012 (VAD)

## 컨텍스트

PRD는 Whisper.cpp 한국어 medium 모델을 제안했다. 2026-05 기준 재검토 결과, large-v3-turbo가 medium 대비 정확도가 높으면서도 속도 우위를 가지며, RTX 5090 환경에서 VRAM 부담이 무의미하다.

## 결정

**faster-whisper (large-v3-turbo 모델)** 를 채택한다.

- CTranslate2 백엔드 → CUDA 가속 + INT8 양자화
- 모델: `large-v3-turbo` (CT2 변환본: `deepdml/faster-whisper-large-v3-turbo-ct2`)
- Python 백엔드 시작 시 즉시 로드, GPU 상시 상주
- 모델 이름은 config.yaml의 `stt.model`로 지정 — 하드코딩 금지

```yaml
# config.yaml
stt:
  model: "large-v3-turbo"
  language: "ko"
  device: "cuda"
  compute_type: "int8_float16"
```

## 결과

### 긍정
- large-v3-turbo는 large-v3 대비 약 216x real-time 속도, 다국어 유지
- medium 대비 정확도 향상, 한국어 인식 품질 개선
- 어댑터 코드 무변경 — 설정 한 줄로 모델 교체 가능

### 부정
- 모델 ~3GB VRAM 상주 (medium ~2GB 대비 증가, 5090에서 무관)
- turbo는 번역(translation) 태스크 미지원 — STT 전용이므로 무관

## 대안

| 후보 | 탈락 사유 |
|---|---|
| faster-whisper medium (초기 결정) | 정확도·속도 모두 turbo에 열위 |
| whisper.cpp | C++ 바인딩, Python 통합 추가 작업 |
| Qwen3-ASR | CJK 특화로 매력적이나 도구 생태계가 faster-whisper만큼 성숙하지 않음 — P2 검토 |
| NVIDIA Parakeet TDT | 영어 전용, 한국어 미지원 |
| distil-whisper | 영어 위주, 다국어 약함 |
