# ADR-026: 운동 종목 — v3 4종 고정 유지

- **상태**: Deferred (2026-06-10)
- **관련 ADR**: ADR-014 (카운팅 엔진), ADR-021 (v4 스코프)

## 컨텍스트

v4-vision §2 는 종목 확장(덤벨·유산소 등) + 사용자 정의 종목을 후보로 적었다. deep interview(§6-7)에서 사용자는 **v3 4종(풀업/푸시업/스쿼트/플랭크) 고정 유지**를 선택했다. v4 의 핵심 역량은 컨디션(ADR-023)·메모리(ADR-025)·플랜(ADR-024)이며, 종목 확장은 우선순위가 낮다.

## 결정

- **v3 4종 고정 유지** — 풀업/푸시업/스쿼트/플랭크.
- 종목 확장·사용자 정의 종목은 **후순위(Deferred)**. 재개 시 별도 ADR.

## 사유

- 종목마다 카운팅 cue·박자 멘트 풀·자세 안내 로직이 필요(ADR-014) — 확장 비용 실재.
- 사용자 정의 종목은 cue·카운팅 방식을 사용자가 정의해야 해 품질 편차·복잡도 큼.
- v4 1차 가치는 "같은 4종을 더 똑똑하게 코칭"(컨디션·플랜·메모리)이지 종목 수가 아님.

## 재개 조건

- v4 핵심(컨디션/플랜/메모리)이 안정된 뒤, 사용자가 특정 종목 추가를 실제로 요구할 때 별도 ADR 로 검토.
